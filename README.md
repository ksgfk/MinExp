# MinExp
学习图形学，OpenGL的工程

## 依赖
尽量使用最少的依赖库
* glfw
  * [https://github.com/glfw/glfw](https://github.com/glfw/glfw)
  * 实在搞不定Native窗口，以及怎么创建OpenGL上下文...（照着抄都抄歪来
* glad
  * [https://glad.dav1d.de](https://glad.dav1d.de)
  * OpenGL函数加载
* stb
  * [https://github.com/nothings/stb](https://github.com/nothings/stb)
  * 不了解图片格式，还是用库吧（
